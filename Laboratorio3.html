¿Qué diferencias y semejanzas hay entre Java y JavaScript?

JavaScript es un lenguaje interpretado, en otras palabras, se lee y traduce línea a línea al mismo tiempo que se ejecuta el programa
Java es un lenguaje compilado: pasa por una JVM (Java Virtual Machine) para ser interpretado y traducido al lenguaje de la «máquina».

¿Qué métodos tiene el objeto Date?

  1. 'getDate()'
  2. 'getDay()'
  3. 'getFullYear()'
  4. 'toTimeString()'
  5. 'setTime()'

¿Qué métodos tienen los arreglos?

  1. 'Array.from()'

  3. 'Array.of()'
  4. 'Array.prototype.concat()'
  5. 'Array.prototype.copyWithin()'
